About Yet Another Chatwork Emoticons Plus

"Yet Another Chatwork Emoticons Plus" (YACEP) is a free Chrome Extension which provides more funny emoticons for Chatwork. It is made by Framgia's Members, and for Framgia's Members.

YACEP is based on the original CWEP (Chatwork Emoticon Plus) made by Nguyen Anh Tien !
Great thanks to him for his hard work :D

Create Your Own Emoticons Data

You can create your own Emoticons data file and then share it with your friends. Of course if you and your friends have the same data, you will see the same emoticons on Chatwork! You can easily create a Emoticons Data file for YACEP by following the below rules:

  • The data file must be written in json format.
  • The data file must have the following data:
    • data_name: Your data name (It should not be updated when you update your data file)
    • data_version: Your data version (It should be updated when you update your data file)
    • emoticons: Emoticons data. It is an array with each element is an emoticon's data. Emoticon's data include "key" (the text that will be replaced by emoticon), "regex" (the regular express for recognizing the key) and "src" (the image's source).
    • You can refer the default emoticons data of YACEP.
  • You need to host your data file on the internet so YACEP can access it!
  • Finally, share the link to your data file with your friends and enjoy your own funny Emoticons!
